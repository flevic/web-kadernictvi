Webová stránka Kadeřnictví Markéta.

Jak spustit webovou stránku:
1)Otevřete složku Web - Kadernictvi.
2)Spusťte zástupce _Start, nebo jakýkoliv jiný .html soubor.
3)Užijte si stránku.

Jak se dostat ke kódu:
1)Otevřete složku Web - Kadernictvi.
2)Klikněte pravým tlačítkem na .html nebo .css soubor.
3)Vyberte v menu možnost otevřít v "váš program na kód například: Visual Studio Code".
4)Také je možné otevřít Visual Studio Code a v něm si vybrat možnost "open folder" a vybrat složku Web - Kadernictvi.
